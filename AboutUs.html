Tourado
User will be able to Sign Up in the system, User can fill out a questionnaire and see recommendations. User can manually plan the tour. User can reserve a room and select a traveling medium.

Tourado will not facilitate Educational Institutes to organize school/college/university trips. For trip booking, the users must be in Pakistan.


In Tourado, users will be able to see popular Pakistani areas along with famous restaurants, hotels, the best traveling mediums & nearby hospitals. Users will be able to plan their tour according to their budget. Packages may vary in prices. After booking a trip, Tourado will calculate total expenses regarding the tour and display it to users. End users should be literate.
